## 移动端购物车
### 技术栈
React + Redux + Router + Immutable + Webpack + ES6 + SCSS + flex

### 开始
1. 安装相关插件 ``npm i``
2. 运行服务 ``npm start``
3. 浏览 ``http://localhost:8080/``

### 演示地址

[demo地址](https://zhao-jun.github.io/react-order-food/#/)（请用chrome手机模式预览）
